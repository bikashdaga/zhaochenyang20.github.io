Readme

  • 这是 zhaochen20 参加计算机系笔记大赛的笔记集合，本人非常荣幸这些可以全体加入 by-pass 集合为计算机系后人铺路
  • 本人的笔记较多，风格各异，部分笔记已经部署在酒井科协 SAST 技能文档，其他部分分享给了少数朋友与计 06 班，私以为质量较高

样张

  • oop 课程笔记( 已部署为技能文档 )

  • 线性代数思维导图

  • 微积分期末复习笔记

  • Linux, Python, JavaScript 笔记( 全部部署在个人博客 )

存储说明

  • 由于笔记数量较多，单独开辟了清华云盘来存储笔记
  • 少部分大一学年的笔记由本人前女友贡献，再次致谢
  • 如若要上传至 by-pass，可以先行剔除笔记里的部分影响阅读的词汇
  • 如若因版权原因造成了不必要的麻烦，烦请告知本人，并将笔记删除
  • 本人所作笔记都有全部本地文档，而 JavaScript 笔记主要是摘抄自技能文档，廖雪峰博客等网络资源，暂时先放入参赛作品，倘若涉及侵权，立刻删除
  • 云盘链接见此处

文件说明

  • caculous 为大一学年的微积分笔记

    1. Calculous Finale 为大一期末复习笔记，写的较为详尽，也做过更改
    2. 其余笔记主要为大一下学期多元微积分内容
    3. 部分笔记为 docx，部分为 md，部分为 pdf

  • oop 为大一下学期的 oop 笔记

    1. 已完全部署至 SAST 技能文档 C++ OOP 部分
    2. 最后的整理工作由计算机系科协 c7w 和 holder 二人整理并格式化，再次致谢

  • linear algebra 为大一上学期的线性代数笔记

    1. 主要为本人所绘制的思维导图

  • complex 为大二上学期的复变笔记

    1. 本人的课程笔记
    2. 期末考试的复习资料

  • 大二下课程目前仍在更新

    1. 所有笔记目前部署在我的个人博客
    2. 目前笔记仍在更新，可查看博客获得最新文章
    3. 笔记内容除了编程基础，还有毛概、体育与科技英语

  • CS 其他为 Linux. Python, JavaScript 等开发笔记

    1. 完全部署在个人博客的 CS 部分和全栈开发部分
