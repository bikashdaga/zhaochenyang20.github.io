小组说明

人员

  • 组长是 zcy 本人~

简单自我介绍下，详细的可以参考我的个人主页

性格非常 easygoing and extrovert，写文档是为了活动成功进行，不会因为不打卡而出锅，也为了显得我很专业！

编程语言主要是 Python(母语), C艹(二外), Linux( 白月光 )，其他略知一二，欢迎讨论！

非常期待和大家一起度过这个学期的活动，其实主要是为了答疑坊积攒稳定客户

  • 大家都有深厚的编程功底，欢迎一起讨论( 不单单是 program )，一同进步鸭！

范围

  • 参考大家的实际需求，回答范围包括但不限于 Python & OOP( C++ ) & Shell( Linux )…
  • 欢迎群里修计算机系 OOP 的同学在群里讨论 OOP 选择题以及阅读本人之前写的 OOP 文档，并对文档提出修改意见( 修改成功本人请饭 )

形式

  • 组内提问，有空就解答( 一般当日解答，没解答麻烦多催一下就好 )
  • 欢迎线下讨论，尤其是配置环境等等问题
  • 线下讨论可以和 zcy 预约答疑坊线下排班，或者约饭( 填写完答疑坊工时后，约饭花费如果在人均 30 RMB 以下，zcy 请客 )

打卡

  • 每次提问完成后，请立即在计算机系的链接里打卡，同时在答疑坊小程序填写线上快捷反馈
  • 如果当天没打卡，我会善意的催促大家
