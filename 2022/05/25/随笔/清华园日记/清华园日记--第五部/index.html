Readme

他想要成长，他必须蜕变。他不愿意再停留在过去，他想把跟自己的过去相连的东西都抛弃。如果他没有告发麦克，没有离开凯西，生活还会像往日一样继续，但他的心已经不在那里了。他已经要离开了，什么都留不住他了。他的灵魂会飞走，尽管他也不知道自己想飞到何处。他唯一知道的是，不能再在这里停留。

成长的蜕变，看似蝴蝶出茧，把丑陋的茧留在原地，自己就能翩翩飞走。但实际上没有那么容易。你和你的过去是紧紧地连在一起的，血肉相连。如果你想要挣脱出去，你一定要撕裂自己的肉，让自己的血流出。当你从过去的自我中爬出来时，你必定已是遍体鳞伤、血肉模糊。这就是成长的残酷现实，这就是青春的丑陋本色。

对抗这个世界很容易，但对抗你的过去、对抗你自己，背叛你的过去、背叛你自己，就没有那么轻松了。

因此，美国作家约翰.厄普代克说；“成长就是背叛，没有其他的道路。没有离开，就没有到达。”

  18. 我从未反对这一观点，中国出兵朝鲜客观上就是武装干涉他国内政。我的观点是，中国出兵朝鲜是有正义性的，因为北朝鲜身后就是东北，而东北是当时中国重工业的火种，直接关系中国存亡。但是美国出兵本质上只是为了抑制苏联在东亚的扩张，是帝国主义争霸的结果，本身和欧洲殖民者在美洲非洲亚洲抢地盘的殖民主义行径底层逻辑是一样的，正义性不可同日而语。

  19. 中国帮助金家建立世袭政权 导致朝鲜人民至今还生活在水深火热中 也算正义？

  20. 谁在装糊涂？你先指责中国抗美援朝时干涉他国内政，我点出中国抗美援朝的正义性并向你询问美国武装干涉的正义性时你又开始避重就轻说都是国家利益。你既然知道都是国家利益一开始还有必要跟我争论？

  21. 北朝鲜的存亡直接关系到中国国家安全，我就想问千里之外的南韩跟美国有什么关系？美国武装干涉的正义性在哪里？我不赞成金家世袭，但是我认为国家安全上来说中国肯定要支持北朝鲜。就像我认为大部分美国人也不能接受沙特的制度，可是这跟美国和沙特成为铁杆盟友没有冲突。

  22. 然后呢？朝鲜族野战师有历史遗留原因按下不表，武装支援跟眼看南朝鲜政府快输了美国亲自下场有可比性？大家都是为了各自的国家利益，只不过你的美国吃相太难看了。从日本起飞都来了，我真的不知道您是蠢还是坏。北朝鲜被美军占领那直接后果是国破家亡，这是主要矛盾。

  23. 讲史实，哈哈哈。

    哪个国家历史上屁股上没粘过屎，但少有人眼中只有屎。

    中国几千年历史，阴暗的事情，令人发指的事情不计其数，

    尽管如此，以宣扬这些事为乐就是带着满满的恶意。

    你跟一群意大利人聊天，人家言必称古老历史，对人类的伟大贡献，就只有你去挑令人扫兴的史实说，是不是还很得意？

  • 我的评价：到达世界最高城——理塘，太美丽了理塘，哎呀，这不丁真吗？还是看看远处的雪山吧，家人们。
